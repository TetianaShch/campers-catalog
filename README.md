# Campers Catalog

Frontend application for browsing and booking campers.

## Tech Stack

- Next.js (App Router)
- TypeScript
- Axios
- TanStack Query

## Features

- Home page
- Campers catalog
- Camper details page
- Filtering and pagination (in progress)

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open http://localhost:3000
in your browser.

API

Data is fetched from:
https://campers-api.goit.study
